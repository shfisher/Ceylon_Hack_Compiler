This project implements LL(1) compiler from sample OOP language (Jack) to sample assembler language (for Hack processor) via Virtual Machine intermediate language. 
Projects 7 & 8 (ex2) perform VM to asm translation,  Projects 10 & 11 (ex4 ex5) - Jack to VM translation, includes tokenizer, analyzer and parser. 
Project 9 (ex3) is sample Jack program - Snake Escape game. 
(See nand2tetris.org for more info)
All work done with command line compiler of Ceylon PL (https://ceylon-lang.org/)
